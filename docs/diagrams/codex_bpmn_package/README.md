# BPMN 2.0 XML files for Codex

These files are prepared in BPMN 2.0 XML format so Codex and BPMN tools can read and edit them as plain text.

Recommended workflow:
1. Put these files in your repository.
2. Keep `AGENTS.md` at the repo root.
3. Ask Codex to modify specific `.bpmn` files.
4. Validate visually in Camunda Modeler or bpmn.io.
