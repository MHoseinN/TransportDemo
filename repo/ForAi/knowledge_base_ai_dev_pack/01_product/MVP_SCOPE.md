# MVP Scope

## Must Have
1. نقش‌ها و دسترسی‌ها
2. مدیریت دسته‌بندی‌ها
3. ثبت سند رسمی
4. تأیید مدیر برای سند سرپرست
5. ویرایش سند با منطق تأیید
6. افزودن نسخه جدید
7. آرشیو نسخه قبلی
8. جستجو و فیلتر
9. جزئیات سند
10. هشدار سند قدیمی
11. پیام روزانه
12. ثبت مطالعه شد
13. API پایه برای نرم‌افزار اصلی
14. Audit Log عملیات مهم

## Should Have
1. Tree View دسته‌بندی‌ها
2. اعلان درخواست تأیید برای مدیر
3. گزارش پایه مطالعه اسناد
4. وضعیت خوانده‌شده/نخوانده‌شده در UI
5. پیوست برای پیام روزانه

## Could Have / Future
1. Full-text search داخل فایل‌های PDF/Word
2. OCR فایل‌های اسکن‌شده
3. Diff خودکار نسخه‌ها
4. پیشنهاد هوشمند سند مرتبط
5. اجبار انتخاب سند هنگام ثبت تماس
