# Open Questions

فعلاً ابهام مسدودکننده وجود ندارد. این موارد می‌توانند بعداً روشن شوند:

1. پشته فنی نهایی تیم چیست؟ `.NET/Vue/SQL Server` فعلاً به‌عنوان پیشنهاد ثبت شده است.
2. احراز هویت از داخل همین سامانه انجام می‌شود یا از SSO/سامانه سازمانی گرفته می‌شود؟
3. حداکثر حجم و فرمت فایل‌های پیوست چه باشد؟ پیشنهاد MVP: PDF, DOC, DOCX, JPG, PNG.
4. آیا جستجوی داخل فایل‌های PDF/Word در MVP لازم است؟ پیشنهاد: فاز ۲.
5. آیا گزارش مطالعه اسناد در MVP باید UI داشته باشد یا فقط API/DB Log کافی است؟ پیشنهاد: Should، نه Must.
