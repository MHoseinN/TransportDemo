# Diagrams

This folder contains Mermaid source diagrams for user flow and BPMN-like flow.

- `03_workflows/USER_FLOW_GENERAL.mmd`
- `03_workflows/BPMN_GENERAL.mmd`

Original Visio VDX and ZIP package are in `99_original_artifacts` if available.

## How to use
- Paste `.mmd` files into Mermaid Live Editor for review.
- Use VDX file in Visio if compatible.
- For formal BPMN, create one Pool for external app and one Pool for Knowledge Base system.
- Knowledge Base system lanes:
  1. Operator/User
  2. Supervisor
  3. Manager/Admin
  4. Knowledge Base System
