## Naming Rules

- Backend commands use imperative names: `CreateVehicleCommand`, `SubmitMissionRequestCommand`.
- Queries use read-oriented names: `GetVehicleByIdQuery`, `SearchMissionRequestsQuery`.
- DTOs are not EF entities.
- API routes are plural and versioned: `/api/v1/vehicles`.
- Frontend feature API files are the only files that know endpoint URLs.
